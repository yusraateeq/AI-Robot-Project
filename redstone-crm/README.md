# Redstone CRM — Frontend

Next.js 16 dashboard for the Redstone AI Voice Bot Agent System.

## Stack

- Next.js 16 (App Router)
- Firebase Authentication (Google & Email/Password)
- Tailwind CSS
- Chart.js
- Framer Motion

## Scripts

```bash
npm run dev      # development
npm run build    # production build
npm run lint     # ESLint
```
